# Project 1

## Not Gate
Not gate built using single Nand Gate.

## And Gate

![And Gate ciruit diagram](Circuit%20Diagrams/and.png)

## Or gate

![Or Gate ciruit diagram](Circuit%20Diagrams/or.png)
